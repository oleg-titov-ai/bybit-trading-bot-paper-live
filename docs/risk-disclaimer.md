## Trading Risk

Cryptocurrency trading is risky.

You can lose money because of:

- market volatility;
- sudden price movements;
- liquidation risk;
- exchange downtime;
- API errors;
- network delays;
- code bugs;
- incorrect configuration;
- excessive leverage;
- poor strategy design.

---

## Automated Trading Risk

Automated trading can amplify mistakes.

A trading bot can:

- open orders faster than a human can react;
- repeat a bad decision many times;
- trade during abnormal market conditions;
- fail if API responses change;
- lose money due to a small configuration error.

---

## Recommended Safety Rules

Start with:

1. Paper trading only.
2. Small demo position sizes.
3. Testnet before mainnet.
4. No leverage unless fully understood.
5. Strict daily loss limits.
6. Manual monitoring.
7. Logs reviewed after every session.
