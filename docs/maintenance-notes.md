# Maintenance Notes

This file keeps lightweight maintenance rules for the trading bot repository.

## Regular Checks

- Keep paper mode as the default safe testing path.
- Review live mode changes carefully before merging.
- Keep `.env.example` free from real credentials.
- Update the risk disclaimer when behavior changes.
- Check logs and screenshots before publishing them.

## Safe Data Rules

Use placeholders for API keys, secrets, account IDs, balances, order IDs, wallet addresses, and exchange responses.

## Release Hygiene

Before publishing a new update, check README accuracy, security notes, setup steps, paper-mode behavior, and changelog entries.
