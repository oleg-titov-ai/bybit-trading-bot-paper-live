# Code of Conduct

## Our Standard

This repository is maintained as a public educational and portfolio project. Discussions should be respectful, practical, and focused on safe software improvement.

Examples of good behavior:

- clear issue reports;
- constructive technical feedback;
- responsible trading-risk discussion;
- safe examples without credentials;
- documentation and testing improvements.

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment or personal attacks;
- publishing API keys, secrets, account data, balances, or private logs;
- spam;
- abusive language;
- misleading financial claims;
- requests for guaranteed profits or unsafe live-trading shortcuts.

## Enforcement

Maintainers may edit, close, delete, or lock issues and pull requests that violate this code of conduct.

## Reporting

Report problems through GitHub issues with a safe, factual description. Do not include exchange credentials, private trading data, screenshots with balances, or personal information.
