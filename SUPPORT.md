# Support

This repository is maintained as a public educational and portfolio project.

## Getting Help

Before opening an issue:

1. read the README;
2. review the risk disclaimer;
3. check the product overview;
4. run the project in paper mode first;
5. avoid live mode until configuration is fully reviewed locally.

## What to Include in an Issue

- short problem description;
- steps to reproduce;
- expected behavior;
- actual behavior;
- Python version;
- whether the problem happens in paper mode or live mode.

## What Not to Include

Do not include API keys, API secrets, real account IDs, balances, wallet data, screenshots with private trading information, or `.env` files.

## Trading Disclaimer

This repository does not provide financial advice. Automated trading can lose money. Paper mode should be used for testing and learning.

## Project Status

This is not a commercial support channel. Issues and improvements are handled as time allows.
